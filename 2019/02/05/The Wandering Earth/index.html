论《流浪地球》内人物的性格真实性

发表于 2019-02-05 | 分类于 Essay | | 阅读次数：

——以下皆为个人看法 或有偏颇 请合理看待——

——内含大量剧透 请注意避雷——

美术设计、音乐、画面、剧情之类的暂不讨论 最让我印象深刻的是几个主角的性格

虽说我也没看过多少电影，但是大部分这类影片的主角总是有一种“不真实”的自我牺牲和奉献意识。例子太多了，随便拉一部美国的满满特效的商业片就行。并非我相信人心本恶或是什么，我真切认为，对于没有经过训练/洗脑/强行灌输的普通人而言，在面对足以危及生命的威胁时，都会是利己自私的。我刚刚说的那类主角，就给我一种英雄人物的刻板印象。他们或许有超能力，或许只是默默无闻的普通人，但他们会成为主角/英雄，正因为他们有着英雄的特性。我并不是不喜欢这类人，只是看多了会有视觉疲劳，颇有些强行灌输价值观的意味。

我看了很多《流浪地球》的差评，其中有不少都说觉得其中的角色“激素泛滥”“聒噪”“不懂事”。私以为不然，这些许多人诟病的要素，是我非常喜欢的。

《流浪地球》不是。

一、主角刘启、妹妹韩朵朵和爷爷韩子昂，在车被强行征用执行救援计划时，刘启是很明显不满的。他本只想去就近的避难所避难，现在却被要求要拖着个火石去更远的杭州，也就是意味着遇到危险的可能性更大。

二、在上海乘坐电梯上顶层的时候，救援队队员要求先拉火石上去，而刘启坚决要求先上人。

三、在大厦快塌陷，拉电梯的牵引绳（？）断了的时候，救援队队员疯狂拉火石，刘启和妹妹只在乎自己的爷爷。在救援队队员刚子为救爷爷去世，而爷爷最终也没逃出去之后，即便知道对方为了救爷爷牺牲了一名战士，刘启和妹妹依旧悲恸欲绝，在当时坚决要求和队员分道扬镳，把车留给他们便好。

四、在听到广播通告杭州地下城塌陷后，还未到达杭州的救援队其中一名队员周倩用枪便打坏了火石，表示不能再让队员牺牲了，现在还是回家吧。

就第三个场景而言，是很多人诟病的一个场景，纷纷表示刘启和朵朵不懂事、幼稚。且不谈刘启，韩朵朵的人设只是个普通的初中女生，无父无母，从小被爷爷抚养长大。此时爷爷去世，如何能不悲痛？如何能保持理性？刘启自然也是同理。而在他们遇到另一队救援队之后，二人的区别便体现出来——听到原救援队的支援请求和杭州地下城塌陷的消息后，较为年长的刘启理智恢复，内心也有些愧疚和后悔，便加入了支援。

第四个场景，也是为很多人诟病的。按理说一个经过训练的战士，是不应当如此随便地打坏重要的国家资源。而，试想，在几个队友都因之牺牲，而作为唯一目标和希望的“救援杭州”也化为泡影，面对三十几万人的死亡，正常人不会产生对自身生命安全性的恐惧吗？打坏火石正是一种希望破灭的崩溃和对死亡的恐惧。

其实，整部影片，哪怕到最后，各个国家的救援队收到全球广播纷纷来帮忙，这个中原因，亦是利己主义的。本是想着趁未灭绝的几天内回家看看家人，也并没有理会朵朵发出的救援信息，而全球广播改变了他们的打算。虽说全球广播并没有明确的态度偏向性，但从空间站的指示来看，他们内心默认该方案是有一定的成功可能性的。故而他们回头，本质来说也是为了自我救援。

听说这部影片是请了北大的心理学教授进行人物性格设计的，虽不知真实性，但总得来说，这部影片的人物性格的确是非常贴合真实情况的（笑）。

2018年度总结
